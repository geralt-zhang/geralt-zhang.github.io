这个网页是基于 Missing 一课的模板改的，这是基于 Jellky 的 Github Page，
仅需关注前端布局，由 Github 自动托管部署，因 Ruby 配置太过麻烦，
本人给它搞了个 Docker 并美化了一下，主要是字体格式和对齐方式，看着舒服多了。

Jekyll 是一个基于 Ruby 的静态网站生成器。
选择它的主要原因是 GitHub Pages 原生支持，代码推送后会自动构建和部署。
它的工作原理是将 Markdown 文件转换为 HTML，生成静态网站。
静态文件的优势是加载速度快，安全性高，因为没有动态内容执行。
Jekyll 需要 Ruby 环境，但版本管理比较复杂。
